# Running Application

## Dependencies:

- Node v16.x
- yarn

## Instructions

- Install yarn dependencies on all folders

```bash
# from project root
yarn && yarn --cwd ./server && yarn --cwd ./web && yarn --cwd ./serverless
```

- Running server and web application in dev

```bash
# from project root
yarn dev
```
